\subsection[The hyperref Package]{The \HyperrefPkg Package}
\begin{frame}[label=more_for_hyperref]{The \HyperrefPkg package}
   \begin{itemize}
      \itemsep8pt
      \item We already created some references (with \blatex{\\tableofcontents}, \blatex{\\ref},~\ldots)
      \item The \HyperrefPkg package automatically transforms these references into clickable hyperlinks
      \item Furthermore it provides useful commands \begin{itemize}
         \item \blatex{\\autoref} is similar to \blatex{\\ref} but tries to identify its target (Figure,~\ldots) and adds the appropriate prefix
         \item \blatex{\\nameref} (due to the \T{\link{https://www.ctan.org/pkg/nameref}{nameref}} package) prints the name of the target (e.g. section name, figure caption,~\ldots)
         \item \blatex{\\hypersetup} controls the color of links,~\ldots
      \end{itemize}
      \item It is advised, to load the \HyperrefPkg package after (almost) all other so it can modify their commands
      \item The \T{\link{https://www.ctan.org/pkg/cleveref}{cleveref}} and \T{\link{https://www.ctan.org/pkg/varioref}{varioref}} packages are interesting as well!
      \item All links within this document are done with \HyperrefPkg (and underlined with the help of \T{\link{https://github.com/EagleoutIce/latex-underline}{latex-underline}})
   \end{itemize}
\end{frame}

\subsection{Including Metadata}
\begin{frame}{Including metadata}
   \begin{itemize}
      \itemsep8pt
      \item You can use \HyperrefPkg to embed metadata into your document
      \item This is usually done for the title, author,~\ldots
      \item You can use \blatex{\\hypersetup} to configure the metadata with \T{pdftitle} and \T{pdfauthor}
      \item Alternatively the package-option \T{pdfusetitle} detects these automatically from the \blatex{\\title} and \blatex{\\author} commands
   \end{itemize}
\end{frame}
